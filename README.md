**overview**
LONG TERM PROJECT(still in progress)
MotionOS is a hands-free PC control system that enables users to control their computer entirely through natural body movements
— hand gestures and eye gaze — using only a standard webcam. No special hardware, no controllers, no physical contact required.

**Gesture Reference**

| Gesture                                    | Action                |
|--------------------------------------------|-----------------------|
| Index finger only, thumb in                | Mouse cursor control  |
| Index finger only, thumb out               | Scroll down           |
| Index + middle, thumb in                   | Disable control       |
| Index + middle, thumb out                  | Scroll up             |
| Index + middle + ring, thumb in            | Single click          |
| Index + middle + ring + pinky, thumb in    | Double click          |
| Index + middle + ring + pinky, thumb out   | Pause media           |
| All fingers folded, thumb up               | Volume up             |
| All fingers folded, thumb down             | Volume down           |
| All fingers folded, thumb neutral          | Play media            |



**Purpose**

- **Accessibility** — Designed for individuals with motor disabilities, paralysis, or conditions such as ALS who cannot operate
 traditional input devices.MotionOS provides an alternative that requires only natural body movement, making PC interaction
 possible for everyone.

- **Medical Use** — In sterile environments such as operation theatres, diagnostic labs, and rehabilitation centers, physical
 contact with devices is not always practical or permitted. MotionOS enables doctors, surgeons, and medical staff to interact
 with screens without touching them.

 - **AI Skill Development** — Built to apply, showcase, and continuously enhance practical knowledge of computer vision and AI
 integration, bridging the gap between academic learning and real-world working systems.
